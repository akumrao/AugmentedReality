# About

  Simple multiplex codec WebRTC, signaling implemented with Socket.IO.

  
# Requirements

* Unity3D 2018.3+
* node.js

# 

* On Windows:

```sh
> server.bat
```
or

* On Mac/Linux

```sh
npm install
node index.js
